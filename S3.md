
## S3 Bucket Creations
```sh
aws s3api create-bucket --bucket bucket-name --region us-east-1
aws s3api list-buckets
```

## S3 Bucket Deletion
```sh
# If there's contents in the S3 bucket
# empty it out with rm command
aws s3 rm s3://the-target-bucket-name
```

## List up the S3 bucket
```sh
aws s3 rb s3://the-target-bucket-name
aws s3api list-buckets --query Buckets[].Name --output table
```

## Upload Contents
```sh
aws s3api put-object --bucket s3-practice-ik-0303 --key hello.txt --content-type plain/txt --body Images/hello.txt
```
