## Create a new s3 bucket

```sh
aws s3 mb s3://checksums-examples
```


## Create a file that will we do a checksum on
```sh
echo "Hello Mars" > myfile.txt
```

## Get a checksu of a file for md5
```sh
md5sum myfile.txt
# <key> myfile.txt
```


## Upload our file nd look at its etg
```sh
aws s3 cp myfile.txt s3://checksums-examples
aws s3api head-object --bucket checksums-example --key myfile.txt
```

## Lets upload a file with a different kind of checksums
```sh
sudo apt-get install rhash
rhash --crc32 --simple myfile.txt # it will generate a key
bundle exec ruby crc.rb
```


```sh
aws s3 put-object \
--bucket="checksums-examples" \
--key="myfilecrc32.txt" \
--body="myfile.txt" \
--checksum-algorithm="CRC32" \
--checksum-crc32="e7c80b87"  # <--- the key generated by rhash
```
